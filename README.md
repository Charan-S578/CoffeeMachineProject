## Coffee Machine Program ☕
A simple Coffee Machine simulation built using Python. The program allows users to order different coffee drinks, processes coin payments, checks ingredient availability, updates resources, and generates reports. It is a beginner-friendly project that demonstrates functions, loops, conditionals, and dictionary handling in Python.



## Features
Multiple coffee options
Espresso
Latte
Cappuccino
Ingredient resource management
Coin processing system
Change calculation
Resource report generation
Transaction validation
Console-based interaction



## Technologies Used
Python 3



## How It Works
User selects a coffee type.
Program checks if enough ingredients are available.
User inserts coins.
Machine verifies payment.
Coffee is served if payment is successful.
Resources are updated automatically.



## Available Commands
Command                             Description 
espresso                            OrderEspresso
latte                               Order Latte
cappuccino                          Order Cappuccino
report                              View remaining resources
off                                 Turn off the machine



## How to Run

Install Python on your system.

Save the file as main.py

Run the program:

python main.py
